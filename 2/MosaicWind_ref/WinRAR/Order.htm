支付了您的注册费以后，您会收到一封电子邮件包含对应您选择的用户名字符串的注册 key 文件。注册时请提供一个可用的 Email 地址，它用来发送给您注册文件。

注册电子邮件会包含必要的指导，只要参照执行。下面我们提供了简要的典型注册手续，但是在注册电子邮件中的指导比这个新，应该有更高的参考价值。

如果您使用 WinRAR 或 RAR/DOS，您应该把 key 文件（rarreg.key）复制到 WinRAR 文件夹中。默认的文件夹是“C:\Program Files\WinRAR”，但是用户能在安装 WinRAR 时改变它。

如果这个 key 是 RAR 压缩文件，为了注册您的 WinRAR，请从压缩文件中解压 key 。

使用 RAR/Unix 和 RAR for OS X 时，您应该把 rarreg.key 复制到您的根目录或下列目录之一： /etc, /usr/lib, /usr/local/lib, /usr/local/etc。 您可以把它改名为 .rarreg.key 或 .rarregkey。如果您愿意，rarreg.key 的名字也是可以的。

现在，WinRAR，RAR for Unix, DOS 和 OS X 使用同样的注册格式，所以您可以将当前 WinRAR 和 RAR 版本的同一个注册码用于多个平台。
